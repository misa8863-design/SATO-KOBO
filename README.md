# SATO KOBO -環の土- Visual LP

沖縄のやちむん作家「SATO KOBO -環の土-」の静的ランディングページです。

## Files

- `index.html`: LP本体
- `style.css`: レスポンシブデザイン
- `script.js`: スクロール表示演出
- `assets/`: セクション順に配置した画像11枚
- `robots.txt`, `vercel.json`: クライアント確認用のnoindex設定

## Links

ECサイト、Instagram、公式LINEのリンクは現在すべて `#` です。公開前に正規URLへ置き換えてください。

## Deploy

このフォルダをGitHubへアップロードし、Vercelでリポジトリを選択してデプロイできます。フレームワーク設定は不要です。
